Visual Studio 2005 já disponível

O Visual Studio 2005, versão final, já está disponível para download aos assinantes MSDN em http://msdn.microsoft.com/subscriptions/.

Visual Studio 2005 significa:

  • Visual C++ 8.0 com melhor otimizador e melhor conformidade com o padrão ISO
  • Profile Guided Optimization
  • C++/CLI
  • compilação para AMD64 (x64) diretamente no Visual Studio
  • Programação em C++ para WindowsCE, PocketPC e Smartphone diretamente no Visual Studio
  • ATL 8, MFC 8
  • runtime do C com verificações de segurança
  • Melhor gerenciamento de propriedades de projetos, para os que como eu, estão a ponto de usar makefiles devido à falta de flexibilidade do Visual Studio no quesito compilação.

E a nova MSDN é compatível com o Firefox, o que é muito bom. Ah, e claro, tem também o .NET Framework 2.0 e o C# 2.0.

Não dá pra comparar o VC com o Builder - até porque, na minha opinião, o propósito da duas IDEs é diferente. O Builder é mais para aplicação com banco de dados ou com uma interface muito trabalhada. O Visual C++ é focado em desempenho e programação mais hardcore, server side e COM, como o que eu faço. E nisso a produtividade é grande (além da IDE ajudar, tem STL e Boost).

Não acho MFC complicado. Dei aulas de MFC para alunos que vieram do VB e eles não tiveram muita dificuldade. Win32 é meio fora de cogitação, apesar de eu já ter feito um discador para um provedor todo em Win32 (eu não conhecia WTL).
